# TASK-0005: knowledgeスキル（ナレッジ管理）

## タスク概要

- **タスクタイプ**: DIRECT
- **推定工数**: 2h

## 実装ファイル

- `templates/skills/knowledge/SKILL.md.template`

## 実装詳細

### ナレッジ保存

1. カテゴリ分類（`{{#knowledge_categories}}` 定義に従う）
2. 重複チェック（タイトル・内容の類似確認）
3. ナレッジフォーマット（knowledge-format.md）に従って整形
4. ストレージの `ナレッジ/{カテゴリ名}/` 配下に保存
5. ナレッジINDEXを更新

### ナレッジ検索

1. キーワードでストレージ検索（`{{storage_search_cmd}}` を使用）
2. `{{#auto_search_rules}}` に基づく自動検索ルールを適用
3. 類似ナレッジを推薦（関連カテゴリも提示）

### カテゴリ構造

`{{#knowledge_categories}}` ループで定義されたカテゴリ:
- デフォルト: 製品・技術仕様（必須）/ 業務フロー・ガイドライン（必須）
- サブカテゴリ対応（`{{#has_subcategories}}` で条件表示）

### referenceファイル

- `knowledge-format.md`: ナレッジページのMarkdownフォーマット定義（フロントマター・本文構造）
