# TASK-0004: docスキル（プリセールス・導入支援文書）

## タスク概要

- **タスクタイプ**: DIRECT
- **推定工数**: 2h
- **依存タスク**: storage-adapter-system TASK-0001 or TASK-0002

## 実装ファイル

- `templates/skills/doc/SKILL.md.template`

## 実装詳細

プリセールス文書（doc）と導入支援文書（engdoc）の両方を担うスキルテンプレート。

### プリセールス文書（docコマンドから呼出）

**prep [顧客名]**
- 案件コンテキストからの顧客情報収集
- Slack・Web検索による最新情報収集
- 商談事前調査レポートの生成

**proposal [顧客名]**
- 案件コンテキスト + ナレッジ（製品情報・成功事例）を参照
- 提案書フォーマットに従って生成
- ストレージのガイドラインページを活用

**estimate [顧客名]**
- 料金体系設定ページを参照
- 顧客要件に基づく見積書生成

### 導入支援文書（engdocコマンドから呼出）

**hearing [顧客名]**
- 標準ヒアリング項目 + 顧客固有項目でヒアリングシート生成

**config [顧客名]**
- 設定確認書（チェックリスト形式）の生成

**testcases [顧客名]**
- テストケースドキュメントの生成

### ストレージ操作

`{{storage_operations}}` 差し込みによりストレージ読み書きを定義。生成文書はストレージの `guidelines/` 配下に保存。
