## 実装詳細

### ルールファイル体系

マイグレーションルールは `templates/migrations/` ディレクトリに配置するファイル群で定義される。

**命名規則**: `v{from}_to_v{to}.md`
- 例: `v1_to_v2.md`（format_version 1 → 2 の変換ルール）

**ルールファイル構成**:
1. 変更概要 — 何が変わったか
2. 変換ルール — 具体的な変換手順（フィールド追加・削除・リネーム・構造変更）
3. 検証条件 — 変換後に満たすべき条件
