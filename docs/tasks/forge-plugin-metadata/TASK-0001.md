## 実装詳細

### plugin.json

```json
{
  "name": "context-stocker-forge",
  "version": "0.8.0",
  "description": "...",
  "author": { "name": "Irii Keita" },
  "keywords": ["context", "stocker", "forge", "generator", "wizard", "template", "backlog", "obsidian", "team", "knowledge", "meta-plugin"]
}
```

`name`, `version`, `description`, `author`, `keywords` の5フィールドのみ。追加フィールドは含めない。

### marketplace.json

`owner`, `metadata`（description/version）, `plugins`配列を含む。

`plugins[0]` に含まれるフィールド: name, source, description, version, author, license(`MIT`), keywords, category(`productivity`)

### バージョニング運用

- コミットごとにバンプ（両ファイルのversionは同じ値を維持）
- **マイナーバージョン（0.x.0）**: テンプレート構造変更、新機能追加、ストレージアダプタ変更
- **パッチバージョン（0.x.y）**: テンプレート文言修正、バグ修正、ドキュメント更新
- コミットメッセージ形式: `v{version}: {変更概要}`
