## 実装詳細

活動ログ管理の統合コマンドテンプレート。3サブコマンドをセクション単位で管理。

### サブコマンド一覧

| サブコマンド | 説明 | 除外フラグ |
|------------|------|----------|
| `daily` | デイリーログ記録（3-Phase並列データ収集） | `{{^excluded_log_daily}}` |
| `weekly` | 週次レポート生成（デイリーログ集約） | `{{^excluded_log_weekly}}` |
| `report` | レポート集約（週次→月次→四半期→年次の階層型） | `{{^excluded_log_report}}` |

各セクションはSkillツールで `{{skill_log_name}}` スキルの対応セクションを呼び出す。

### 特記事項（最適化）

- **daily**: データ収集をPhase 1-3構造で並列化（Slack検索・カレンダー・Backlog課題を並列取得）（最適化B）
- **weekly**: デイリーログの集約と重要度ティアによる絞り込みを実施
