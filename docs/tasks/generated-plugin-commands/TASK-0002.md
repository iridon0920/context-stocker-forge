## 実装詳細

### {pre}-knowledge-save

- ナレッジの保存コマンド
- Skillツールで `{{knowledge_skill_reference}}` スキルを呼び出し
- スキルの「ナレッジ保存」セクションを実行
- excluded_commandsに `knowledge-save` が含まれる場合はファイル自体を生成しない

### {pre}-knowledge-search [キーワード]

- ナレッジの検索コマンド
- `$ARGUMENTS` に検索キーワードを受け取り、Skillツールで `{{knowledge_skill_reference}}` スキルを呼び出し
- スキルの「ナレッジ検索」セクションを実行
- excluded_commandsに `knowledge-search` が含まれる場合はファイル自体を生成しない
