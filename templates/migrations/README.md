# マイグレーションルール

このディレクトリには、コンテキスト・ナレッジページのフォーマットバージョンアップ時に適用する変換ルールを配置する。

## ファイル命名規則

```
v{from}_to_v{to}.md
```

例: `v1_to_v2.md` — format_version 1 → 2 の変換ルール

## ルールファイルの構成

各ファイルには以下を記述する:

1. **変更概要** — 何が変わったか
2. **変換ルール** — 具体的な変換手順（フィールド追加・削除・リネーム・構造変更）
3. **検証条件** — 変換後に満たすべき条件

## 使用タイミング

`/{{product_prefix}}-admin migrate` コマンド実行時、SKILL.md のマイグレーションモード手順 (Step 7) で参照される。
